# ASADA Lisboa Frontend

## 🛠️ Getting Started

### Prerequisites
- Visual Studio Code
- Angular 20   
- Node 22
- NPM 11

### Installation
1. Clone repository
```sh
git clone https://github.com/ejbpz/asada-lisboa-frontend.git
```

2. Install npm packages
```sh
npm install
```

3. Run the project
```sh
ng serve
